Personal Portfolio Website

Live Website: https://drabdulkarimalmuhammad.vercel.app/

Modern, clean, and fully responsive personal portfolio website for Dr. Abdulkarim Al-Muhammad – University Lecturer, PhD in Control & Automation Engineering, Blockchain & Web3 Expert.

## Features

- Beautiful modern design with smooth gradients and glass-morphism touches  
- 100% responsive on all devices  
- Full bilingual support: English (LTR) & Arabic (RTL) with automatic direction switching  
- One-click language toggle with preference saved in localStorage  
- Smooth scroll-reveal animations for cards  
- SEO-optimized with Open Graph & meta tags  
- Fast loading, no framework, pure vanilla code  
- Clean, well-commented, and maintainable codebase  
- Ready for future dark mode implementation  

## Technologies Used

- HTML5  
- CSS3 (Custom Properties, Flexbox, Grid, Cubic-Bezier transitions)  
- Vanilla JavaScript (ES6+)  
- Google Fonts – Cairo  
- Font Awesome 6.5.2  
- Lightweight custom i18n system (no external libraries)
